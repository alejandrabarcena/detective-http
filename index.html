🕵️‍♀️ El detective del servidor

Este proyecto analiza cómo los navegadores y servidores web se comunican usando respuestas HTTP. Usé herramientas como DevTools y plataformas online para observar códigos, encabezados, redirecciones y errores como toda una QA pro.

📚 Proyecto educativo

🌐 Sitios analizados

  • 🇩🇪 Deutsche Welle
  • 🇬🇧 BBC Learning English
  • 🇫🇷 TV5Monde

🛠️ Herramientas utilizadas

  • DevTools del navegador (pestaña Network)
  • httpstatus.io
  • reqbin.com
  • webhook.site

🔎 Qué se analizó

  • Códigos de estado como 200, 301, 404
  • Redirecciones automáticas
  • Encabezados HTTP como content-type, cache-control, etc.
  • Respuestas ante errores (404 personalizados)
  • Políticas de caché

💡 Conocimientos obtenidos

  • Interpretar respuestas del servidor sin escribir código
  • Analizar red y rendimiento desde el navegador
  • Detectar errores como tester de calidad web
